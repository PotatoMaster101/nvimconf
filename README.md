# Neovim Configuration
Neovim initialisation scripts.

## Usage
For **Linux**, run with shell:
```
$ git clone https://github.com/PotatoMaster101/nvimconf
$ cd nvimconf && /bin/sh init.sh
$ cd .. && rm -rf nvimconf    # clean up, optional
```

For **Windows**, run with PowerShell:
```
> git clone https://github.com/PotatoMaster101/nvimconf
> .\nvimconf\init.ps1
> Remove-Item -Force -Recurse .\nvimconf\       # clean up, optional
```
